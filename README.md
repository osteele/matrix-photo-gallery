# Matrix Photo Gallery

\[Under construction]

A photo gallery for photos from a Matrix room.

Developed as an experimentation platform for [Dinacon
2018](https://www.dinacon.org).

## Develop

Run the back end:

```shell
$ pipenv install
$ pipenv run server
```

Run the front end:

```shell
$ cd front
$ yarn install
$ yarn start
```
